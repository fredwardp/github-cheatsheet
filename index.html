Branches anlegen und wechseln

  1. 1. git branch - Zeigt alle vorhandenen branches an und unter “\* branch“, wo man sich gerade befindet.
  2. 2. git branch "Branch_Name" - Legt einen neuen Branch an.
  3. 3. git checkout "Branch Name" - Wechselt zu einem neuen Branch
  4. 4. git checkout -b "Branch Name" - Legt einen neuen Branch "Branch Name ´" an und wechselt dahin
  5. 5. git branch -d "Branch Name" - Löscht den branch "Branch Name"

Ablauf eines Merges (Zusammenführung zweier Branches)
